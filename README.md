# Ejecutar en desarrollo:
1. Clonar repositorio
2. Ejecutar:
    ```
    npm install
    ```
3. Tener Nest CLI instalado:
    ```
    npm i -g @nestjs/cli
    ```
4. Levantar BD
    ```
    docker-compose up -d
    ```

## Stack usado
* MongoDB
* NestJS


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
